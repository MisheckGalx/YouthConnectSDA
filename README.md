# Clayville SDA Church Website

A premium, professional church website with admin panel for easy content management.

## Features

- ✅ Premium Apple/Nike-style design
- ✅ Hero slider with smooth transitions
- ✅ Events & Sermons management
- ✅ Admin panel for weekly updates
- ✅ Responsive mobile design
- ✅ Clean White, Gray, Maroon, Black color scheme
- ✅ Easy 5-minute weekly maintenance

## Installation

### Prerequisites

- Python 3.8 or higher
- pip (Python package manager)

### Setup Steps

1. **Clone or download this project**

2. **Install dependencies**
```bash
   pip install -r requirements.txt
```

3. **Initialize the database**
```bash
   python database.py
```

4. **Run the application**
```bash
   python app.py
```
